Welcome!

This site lets you encrypt and decrypt text using One Time Pad cipher

This very simple web site lets you encrypt and decrypt using One Time Pad (OTP) encryption which is an encryption algorithm that offers perfect secrecy via a single use key.

Disclaimer

Do not use this site for actual important encryption!
This is only a simple implementation made for learning purposes.
If your life depends on encrypting a message, maybe think about using some other tool :)

Pros

Pros of using One Time Pad for encryption

  • It offers perfect secrecy: as long as an attacker doesn't have the key associated with the cyhertext, it can't decode the message in any way but by trying and guess the key.
  • Very simple implementation and usage (it encrypts a message by executing simple bitwise XOR between message and key)

Cons

Drawbacks of One Time Pad for encryption

  • You have to find a secret and secure way to exchange the key with the receiver.
  • The key has to have at least the same length as the message, meaning that if you have a very large text, the key will be large too.
